# Dodge Manager Pro

ドッジボールチームの運営を支援する、Windowsタブレット（10インチ・横画面）向け統合アプリケーションです。
高度なアクション定義機能により、チーム独自の指標で試合を記録・分析できます。

## 機能概要

### 1. 試合記録 (Game Record)
- **動的アクションボタン**: 設定に基づき「アタック成功」「アタック失敗」などのボタンを自動生成。
- **結果別詳細記録**: 「成功時」と「失敗時」で異なる詳細項目（コース、ミス原因など）を選択可能。
- **リアルタイム記録**: 試合タイマー（タイム機能付き）と連動したプレーログ記録。
- **データ連携**: 登録済みの選手（背番号・コートネーム優先）をベンチメンバーとして自動ロード。

### 2. チーム管理 (Team Management)
- **名簿管理**: 選手情報の登録・編集。背番号やコートネームの管理に対応。
- **表示カスタマイズ**: 項目ごとのソートや表示/非表示の切り替え。
- **CSV連携**: チームデータのインポート・エクスポート。

### 3. 統合設定 (Settings)
- **アクション定義**:
  - 「成功」「失敗」の記録有無を個別に設定可能。
  - 結果に応じた詳細項目（サブアクション）の定義。
- **試合環境設定**: 試合時間やボタン配置（列数）の調整。

## 開発環境

- **Framework**: Flutter
- **Target Platform**: Windows (10/11)
- **Database**: `sqflite_common_ffi` (SQLite v4)
  - チーム・選手データ
  - 試合結果・ログデータ（結果ステータス付き）
  - アクション定義（構造化サブアクション）

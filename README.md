# Dodge Manager Pro

Dodge Manager Proは、ドッジボールなどのチームスポーツ向けに設計された、試合のリアルタイム記録・分析・チーム管理を行うためのFlutterアプリケーションです。デスクトップ（Windows, macOS, Linux）およびモバイル環境での動作をサポートしています。

## 主な機能

### 1. 試合記録 (Match Recording)
* **リアルタイムログ**: 試合中のアクション（シュート、パス等）を選手ごとに素早く記録。
* **タイマー管理**: 試合時間の計測、一時停止、終了処理に対応。
* **選手ステータス管理**: 「コート」「ベンチ」「欠席」のステータスをタップ操作で切り替え。
* **試合詳細の保存**: スコア、対戦相手、会場、試合種別（公式戦・練習試合等）をデータベースに保存。

### 2. データ分析 (Analysis)
* **統計集計**: 期間（年・月・日）や特定の試合ごとの成績を集計。
* **多様なビュー**: 集計データ、プレーログ、試合情報、出場メンバーをタブで切り替えて確認。
* **PDF出力**: 集計結果を印刷用レイアウトで生成し、PDFとしてエクスポート可能。

### 3. チーム・名簿管理 (Team Management)
* **カスタムスキーマ**: チームごとに必要な項目（背番号、ポジション等）を自由に定義可能。
* **名簿インポート/エクスポート**: CSV形式でのデータ一括登録・出力に対応。
* **複数チーム対応**: アプリ内で複数のチームを切り替えて管理。

### 4. 設定とデータ保護 (Settings & Backup)
* **アクション定義**: 記録するアクションや詳細（サブアクション）をカスタマイズ。
* **バックアップ機能**: 全データをファイルとして保存し、必要時に復元可能。
* **データベース**: ローカルSQLiteを使用し、オフラインでも全機能を利用可能。

## 技術スタック

* **Framework**: Flutter
* **State Management**: Riverpod (flutter_riverpod / hooks_riverpod)
* **Database**: sqflite / sqflite_common_ffi
* **Code Generation**: Freezed, JsonSerializable
* **UI**: Material 3, Google Fonts

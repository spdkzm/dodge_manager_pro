# Dodge Manager Pro

ドッジボールチームの運営を支援する、Windowsタブレット（10インチ・横画面）向け統合アプリケーションです。
チーム管理から試合記録、データの永続化までを一つのプラットフォームで提供します。

## 機能概要

### 1. 試合記録 (Game Record)
- **データ連携**: 名簿管理で登録した選手（背番号・氏名）を自動ロードし、ベンチメンバーとして展開。
- **動的アクション**: 設定画面で定義したアクション（アタック、キャッチ等）をデータベースから読み込んでボタンを生成。
- **DB保存**: 試合結果および詳細ログをSQLiteデータベースに永続保存。
- **履歴閲覧**: 保存された試合記録の詳細閲覧。

### 2. チーム管理 (Team Management)
- **名簿管理**: 選手情報の登録・編集・削除。背番号管理に対応。
- **CSV連携**: チームデータのインポート・エクスポート機能（バックアップ用途）。
- **スキーマ設定**: 管理項目の自由なカスタマイズ。

### 3. 統合設定 (Settings)
- **アクション定義**: 試合で使用するアクションボタンの名称や詳細項目を自由に編集・保存。
- **試合環境設定**: 試合時間やボタン配置（列数）の調整。
- **一元管理**: チーム管理やデータ入出力もここからアクセス可能。

## 開発環境

- **Framework**: Flutter
- **Target Platform**: Windows (10/11)
- **Database**: `sqflite_common_ffi` (SQLite)
  - チーム・選手データ
  - 試合結果・ログデータ
  - アクション定義・設定

# Dodge Manager Pro

ドッジボールチームの運営を支援する、Windowsタブレット（10インチ・横画面）向け統合アプリケーションです。
「試合記録」と「名簿管理」のデータを連携させ、スムーズなチーム運営を実現します。

## 機能概要

### 1. 試合記録 (Game Record)
- **データ連携**: チーム管理機能で登録された選手（背番号付き）を自動で読み込み、ベンチメンバーとして表示。
- **リアルタイム記録**: 試合タイマーと連動したプレーログ記録。
- **ベンチワーク**: コート・ベンチ・欠席の選手配置管理。

### 2. チーム管理 (Team Management)
- **名簿管理**: 選手の詳細情報管理。新しく「背番号」フィールドをシステム標準としてサポート。
- **カスタマイズ**: 管理項目の追加・変更（スキーマ設定）。
- **データ連携**: CSV形式でのインポート・エクスポート。
- **データベース**: SQLiteによる堅牢なデータ保存。

## 開発環境

- **Framework**: Flutter
- **Target Platform**: Windows (10/11)
- **Database**:
  - `sqflite_common_ffi`: 名簿データおよびチーム設定
  - `shared_preferences`: 試合ログの一時保存（将来的にSQLiteへ移行予定）

# 🏀 Dodge Manager Pro

**ドッジボール チーム管理・試合記録・データ分析アプリ**

Dodge Manager Pro は、ドッジボールチームのコーチやマネージャー向けに設計された、高機能なスコアブック＆データ分析ツールです。
リアルタイムでの試合記録から、詳細なスタッツ分析、名簿管理までをワンストップで提供します。

## ✨ 主な機能

### 1. チーム・名簿管理
* **マルチチーム対応:** 複数のチームを作成し、切り替えて管理可能。
* **柔軟な名簿設定:** 背番号、名前だけでなく、学年や連絡先など、必要な項目を自由にカスタマイズ（スキーマ設定）。
* **CSV連携:** 既存の名簿データをCSVで一括インポート・エクスポート可能。電話番号や背番号の「先頭の0」落ち対策済み。

### 2. 試合記録 (Real-time Recording)
* **直感的な入力:** タップ操作で「誰が・何をして・どうなったか（成功/失敗）」を素早く記録。
* **試合種別:** 「大会」「練習試合」「練習」のタグ付けが可能。
* **状態維持:** アプリを閉じても、入力中のベンチメンバーやタイマー状態を保持。連続した試合記録もスムーズ。
* **アクションカスタマイズ:** チームの方針に合わせて、ボタンの配置やアクション名（アタック、キャッチ、パスミス等）を自由に設定可能。

### 3. データ分析 (Analytics)
* **多角的な集計:** 全期間、年、月、日、試合ごとのスタッツを自動計算。
* **詳細ログビュー:** スタッツだけでなく、時系列のプレーログ（タイムライン）も確認可能。
* **高度なフィルタリング:** 「大会のみ」「練習試合のみ」といった試合種別での絞り込みに対応。
* **データ修正:** 記録ミスがあっても安心。後から「誰の」「何のアクションか」「時間」「日付」を修正可能。
* **CSVエクスポート:** 集計結果をCSVで出力し、Excel等で二次利用が可能。

## 🛠️ 開発環境とビルド方法

このプロジェクトは **Flutter** で開発されています。
状態管理に `Riverpod`、データベースに `sqflite`、コード生成に `freezed` を使用しています。

### 必須要件
* Flutter SDK (3.x以上)
* Dart SDK
